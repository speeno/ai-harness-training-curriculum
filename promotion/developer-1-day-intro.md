# 개발자 트랙 1일 과정 소개

## 한 줄 소개

Cursor 기반 하네스에서 에이전트, 스킬, 오케스트레이터, Task, `AGENTS.md`, 검증 개념을 하루 안에 실습으로 연결하는 개발자 입문 과정입니다.

## 누구에게 맞나요

- 소프트웨어 엔지니어, 플랫폼·DevEx, 데이터·자동화 엔지니어
- 리포지토리, Git, IDE, 스크립트 기반 업무에 익숙한 실무자
- AI 에이전트를 사내 개발 워크플로에 안전하게 연결하고 싶은 팀

## 교육 목적

1일 과정의 목적은 에이전트 하네스를 구성하는 핵심 요소의 역할을 구분하고, 작은 예제로 실행 흐름을 이해하는 것입니다. 수강자는 `SKILL.md`를 재사용 가능한 지침 단위로 작성하고, 오케스트레이터 스킬과 Task·서브에이전트의 위임 경계를 체험하며, `AGENTS.md` 포인터와 기본 검증 절차를 연결합니다.

## 핵심 커리큘럼

| 시간대 | 주제 | 핵심 활동 |
| --- | --- | --- |
| 오전 | 하네스 스택, `SKILL.md`, 오케스트레이터 개념 | 스킬 구조와 트리거, 위임 경계 이해 |
| 오후 | Task·서브에이전트, `AGENTS.md`, 검증 | 분해 기준, 포인터 작성, 스모크 체크 설계 |

## 얻을 수 있는 결과

- 연습 레포 또는 브랜치에 `SKILL.md` 1개 이상 작성
- 오케스트레이터 스킬 초안 1개
- `AGENTS.md`에 스킬 또는 빌드 명령 포인터 추가
- Task 분해 노트: 서브에이전트 경계와 금지 사항 정리
- 스모크 절차와 기대 로그 키워드가 담긴 검증 메모

## 추천 활용 상황

- 개발팀이 AI 에이전트 기반 자동화의 기본 개념을 빠르게 맞추고 싶을 때
- 3일 또는 1주 심화 과정 전에 공통 실습 경험을 만들고 싶을 때
- 비개발 트랙에서 나온 업무 시나리오를 기술 요구사항으로 변환하기 시작할 때

## 준비물

노트북, Cursor, Git, 사내 샘플 레포 접근 권한, 팀 표준 런타임(Node 또는 Python 등), 선택적으로 스모크 스크립트 실행 환경을 준비합니다.
