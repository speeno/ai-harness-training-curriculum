## 4. 샘플 데이터 준비 기준

샘플 데이터는 실제 업무와 유사해야 하지만, 다음 정보는 제거하거나 가명 처리해야 합니다.

- 개인정보
- 고객 식별 정보
- 계약 금액 또는 민감한 재무 정보
- 계정, 토큰, 비밀번호
- 내부 기밀 문서 원문
- 외부 반출이 금지된 자료

권장 샘플 유형:

| 유형 | 예시 |
| --- | --- |
| 정상 샘플 | 일반적인 업무 요청서, 회의록, 보고서 |
| 경계 샘플 | 일부 정보가 누락된 자료 |
| 오류 샘플 | 형식이 깨진 자료, 잘못된 입력 |
| 민감정보 포함 샘플 | 가상의 개인정보가 포함된 테스트 자료 |
