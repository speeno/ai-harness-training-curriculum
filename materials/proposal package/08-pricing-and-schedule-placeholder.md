# 일정 및 비용 제안

> 이 문서는 고객사 또는 내부 정책에 따라 조정하는 플레이스홀더입니다.

## 1. 제안 범위

본 제안의 범위는 하네스(Harness) · Agent Skills 기반 기업 AI 교육의 설계, 운영, 실습 지원, 산출물 리뷰입니다.

선택 가능한 과정:

- 1일 도입 과정
- 3일 파일럿 설계 과정
- 1주 조직 표준화 과정
- 비개발자 트랙
- 개발자 트랙
- 혼합 과정

---

## 2. 과정별 일정안

| 과정 | 기간 | 권장 인원 | 운영 방식 | 비고 |
| --- | --- | --- | --- | --- |
| 1일 과정 | 1일, 6~7시간 | 10~25명 | 강의 + 실습 + 발표 | 도입 워크숍 |
| 3일 과정 | 3일 연속 또는 주 1회 × 3주 | 8~20명 | 워크숍 + 산출물 리뷰 | 파일럿 설계 |
| 1주 과정 | 5영업일 | 6~15명 | 집중 랩 + 리뷰 + 최종 발표 | 조직 표준화 |

---

## 3. 운영 방식

| 운영 방식 | 설명 |
| --- | --- |
| 오프라인 | 강의장 또는 사내 교육장 진행 |
| 온라인 | 화상회의와 협업 문서 기반 진행 |
| 하이브리드 | 핵심 강의는 온라인, 실습과 리뷰는 오프라인 |
| 집중형 | 연속 일정으로 빠르게 산출물 완성 |
| 분산형 | 주 1회 또는 격일 운영으로 현업 병행 |

---

## 4. 투입 인력

| 역할 | 주요 업무 | 필요 여부 |
| --- | --- | --- |
| Lead Instructor | 전체 강의 진행, 핵심 개념 설명 | 필수 |
| Facilitator | 실습 지원, 조별 피드백 | 권장 |
| Technical Coach | 개발자 트랙 실습 지원 | 개발자 과정 필수 |
| Domain Reviewer | 업무 도메인 검토 | 고객사 제공 권장 |
| Security Reviewer | 민감정보·보안 기준 검토 | 3일 이상 권장 |

---

## 5. 비용 산정 기준

최종 비용은 다음 요소에 따라 산정됩니다.

- 교육 기간: 1일 / 3일 / 1주
- 트랙 구성: 비개발자 / 개발자 / 혼합
- 참가자 수
- 사전 맞춤화 범위
- 샘플 데이터 및 도메인 시나리오 제작 범위
- 강사 및 보조 퍼실리테이터 투입 규모
- 교육 후 산출물 리뷰 포함 여부
- 후속 파일럿 컨설팅 포함 여부

제안 금액:

| 항목 | 금액 | 비고 |
| --- | --- | --- |
| 교육 설계 | [금액 입력] |  |
| 강의 운영 | [금액 입력] |  |
| 실습 지원 | [금액 입력] |  |
| 자료 맞춤화 | [금액 입력] |  |
| 후속 리뷰 | [금액 입력] |  |
| 합계 | [금액 입력] |  |

## 6. 고객사 준비사항

| 준비 항목 | 설명 | 담당 |
| --- | --- | --- |
| 참가자 확정 | 부서, 역할, 기술 수준 확인 | 고객사 |
| 샘플 데이터 준비 | 비식별 업무 샘플 | 고객사 |

| 파일럿 후보 업무 선정 | 교육 전 1~3건 | 고객사 |
| 교육 장소 또는 접속 링크 | 오프라인/온라인 환경 | 고객사 |

## 7. 선택 옵션

| 옵션 | 설명 |
| --- | --- |
| 사전 인터뷰 | 교육 전 업무 후보와 조직 상황 파악 |
| 맞춤 시나리오 제작 | 고객사 업무에 맞춘 실습 시나리오 구성 |
| 산출물 리뷰 | 교육 후 산출물에 대한 전문가 피드백 |
| 파일럿 컨설팅 | 교육 후 파일럿 실행 지원 |
| 보안 리뷰 세션 | 민감정보, 로그, 승인 기준 별도 점검 |
| 경영진 브리핑 | 교육 결과와 파일럿 후보 보고 |

## 8. 가정 및 제외 범위

### 기본 가정

- 교육 중 사용하는 데이터는 비식별 처리되어야 합니다.
- 고객사는 교육에 필요한 도구 접근 권한을 사전에 제공합니다.
- 교육은 자동화 시스템 완성보다 산출물 작성과 파일럿 준비를 목표로 합니다.

### 제외 범위 예시

- 실제 운영 시스템 개발
- 고객사 내부 보안 승인 대행
- 프로덕션 배포
- 민감정보 원본 데이터 처리
- 외부 시스템 연동 개발
- 장기 운영 대행
