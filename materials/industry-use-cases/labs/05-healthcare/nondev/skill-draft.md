# 업무 매뉴얼형 Skill 초안: 병원 예약 변경·민원 분류

## 목적

예약 변경 요청과 민원을 유형별로 분류하고 에스컬레이션 초안을 만든다.

## 입력 자료

- 예약 변경 요청 목록
- 민원 메모
- 진료과별 운영 기준

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 내용을 분류합니다.
3. 근거가 부족하거나 승인이 필요한 항목을 `확인 필요`로 표시합니다.
4. 사람이 검토할 수 있는 표와 요약을 작성합니다.
5. 최종 공유 전 담당자가 표현과 민감정보를 확인합니다.

## 금지 사항

- 입력에 없는 사실을 만들어내지 않습니다.
- 의학적 판단을 하지 않는다. 환자 식별 정보는 입력에도 샘플값만 사용하고 출력에서 제거한다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
