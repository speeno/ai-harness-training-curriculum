# 비개발자 Skill 초안 템플릿

## 목적

이 Skill이 어떤 반복 업무를 어떤 산출물로 정리하는지 2~3문장으로 적습니다.

## 입력 자료

- 입력 1
- 입력 2
- 입력 3

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 분류합니다.
3. 확인이 필요한 항목을 별도 표시합니다.
4. 사람이 검토할 수 있는 표와 요약으로 정리합니다.

## 금지 사항

- 원본에 없는 내용을 사실처럼 쓰지 않습니다.
- 승인·법무·의료·세무·품질 판단을 자동 확정하지 않습니다.
- 개인정보를 공유본에 포함하지 않습니다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
