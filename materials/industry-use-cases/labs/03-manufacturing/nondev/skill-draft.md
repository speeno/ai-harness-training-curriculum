# 업무 매뉴얼형 Skill 초안: 제조 품질 이슈 로그·생산 회의록

## 목적

품질 이슈 로그와 생산 회의 메모를 통합해 조치 항목과 확인 필요 사항을 정리한다.

## 입력 자료

- 라인별 품질 이슈 로그
- 생산 회의록
- 설비 점검 메모

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 내용을 분류합니다.
3. 근거가 부족하거나 승인이 필요한 항목을 `확인 필요`로 표시합니다.
4. 사람이 검토할 수 있는 표와 요약을 작성합니다.
5. 최종 공유 전 담당자가 표현과 민감정보를 확인합니다.

## 금지 사항

- 입력에 없는 사실을 만들어내지 않습니다.
- 원인은 후보로만 표현한다. 안전·품질 인증 관련 항목은 반드시 human_review_required로 표시한다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
