# 업무 매뉴얼형 Skill 초안: 회계·세무 증빙 검토·월마감 메모

## 목적

월마감 전 증빙 목록을 검토해 누락·불일치·고객 확인 필요 항목을 정리한다.

## 입력 자료

- 증빙 목록 CSV
- 전월 마감 메모
- 고객사 확인 요청 메모

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 내용을 분류합니다.
3. 근거가 부족하거나 승인이 필요한 항목을 `확인 필요`로 표시합니다.
4. 사람이 검토할 수 있는 표와 요약을 작성합니다.
5. 최종 공유 전 담당자가 표현과 민감정보를 확인합니다.

## 금지 사항

- 입력에 없는 사실을 만들어내지 않습니다.
- AI 산출물은 검토 보조이며 세무 판단을 확정하지 않는다. 금액·사업자 정보는 샘플 값만 사용한다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
