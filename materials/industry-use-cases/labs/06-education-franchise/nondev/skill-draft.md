# 업무 매뉴얼형 Skill 초안: 교육·프랜차이즈 지점 운영 리포트·교육 자료 표준화

## 목적

지점 방문 점검 메모를 정리해 운영 리포트와 교육 자료 개선 초안을 만든다.

## 입력 자료

- 지점 방문 점검 메모
- 교육 자료 개선 요청
- 본부 운영 기준

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 내용을 분류합니다.
3. 근거가 부족하거나 승인이 필요한 항목을 `확인 필요`로 표시합니다.
4. 사람이 검토할 수 있는 표와 요약을 작성합니다.
5. 최종 공유 전 담당자가 표현과 민감정보를 확인합니다.

## 금지 사항

- 입력에 없는 사실을 만들어내지 않습니다.
- 개인명·학생 정보는 사용하지 않는다. 지점 평가는 관찰 사실과 개선 후보를 구분한다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
