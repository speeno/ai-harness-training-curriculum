# Pilot Design 예시

## 1. Pilot Scope

`교육·프랜차이즈 지점 운영 리포트·교육 자료 표준화` 업무를 1개 팀 또는 1개 SKU·고객군·지점 범위에서 4주간 검증합니다.

## 2. Success Criteria

- 산출물이 지정 경로에 생성됩니다.
- 사람 검토 항목이 누락되지 않습니다.
- 반복 업무 시간이 줄어드는지 정성 피드백을 수집합니다.

## 3. Risk Register

| Risk | Mitigation |
| --- | --- |
| 입력 누락 | `input_missing`로 중단하고 담당자에게 요청 |
| 판단 과잉 | `human_review_required` 표시 |
| 민감정보 | 비식별 후 재실행 |

## 4. Review Cadence

주 1회 결과물과 Smoke 결과를 함께 검토합니다.
