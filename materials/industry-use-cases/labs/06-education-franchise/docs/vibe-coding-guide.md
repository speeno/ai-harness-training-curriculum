# Vibe Coding Guide

## 1일 과정

1. `prompts/00-lab-orientation.md`로 Agent에게 lab 목표를 설명합니다.
2. `prompts/01-scaffold-workspace.md`로 입력 파일과 출력 위치를 확인합니다.
3. `prompts/02-refine-skill.md`로 `skills/store-ops-training-report/SKILL.md`를 현재 샘플 기준에 맞게 점검합니다.
4. `prompts/03-run-output.md`로 `workspace/out/store-ops-training-report-draft.md`를 만듭니다.
5. `prompts/04-smoke-verify.md`로 결과를 검증합니다.

## 3일 과정

- `docs/task-breakdown.md`를 사용해 파일럿 Task를 나눕니다.
- `skills/orchestrator/SKILL.md`로 입력 확인, 산출물 생성, 검증, 수정 단계를 연결합니다.
- `workspace/out/pilot-design.md`를 추가 작성합니다.

## 1주 과정

- 반복 운영을 위한 `workspace/out/runbook.md`와 `workspace/out/roadmap.md`를 만듭니다.
- `교육·프랜차이즈` 업무에서 다음 확장 후보를 검토합니다: 신규 지점 온보딩 체크리스트, 교육 자료 변경 이력 관리, 월간 지점 운영 베스트 프랙티스.
