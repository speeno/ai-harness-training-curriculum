# Prompt 02: Refine Skill

## 목적

현재 샘플 데이터에 맞춰 Skill의 절차와 출력 계약을 점검합니다.

## Agent 지시문

`skills/store-ops-training-report/SKILL.md`와 `docs/industry-constraints.md`를 읽고, `지점 방문 점검 메모를 정리해 운영 리포트와 교육 자료 개선 초안을 만든다.`에 필요한 분류 기준·확인 필요 항목·출력 구조가 충분한지 검토하세요. 부족한 점이 있으면 `workspace/out/skill-review-notes.md`에 일반화된 개선안을 작성하세요. 단순히 이번 샘플만 맞추는 임시방편은 피하세요.
