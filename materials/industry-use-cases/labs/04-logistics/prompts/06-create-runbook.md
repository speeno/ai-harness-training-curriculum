# Prompt 06: Create Runbook

## 목적

1주 과정에서 파일럿을 반복 운영할 수 있도록 Runbook 초안을 만듭니다.

## Agent 지시문

`docs/business-scenario.md`, `docs/industry-constraints.md`, `skills/orchestrator/SKILL.md`, `workspace/out/claim-exception-brief-draft.md`를 읽고 `workspace/out/runbook.md`를 작성하세요.

Runbook에는 다음 섹션을 포함하세요.

1. 운영 목적
2. 입력 수집 주기와 담당자
3. Agent 실행 순서
4. 사람 검토·승인 기준
5. 예외 처리
6. 보관·공유 규칙
7. 다음 회차 개선 항목

입력에 없는 조직 정책은 만들지 말고 `확인 필요`로 표시하세요.
