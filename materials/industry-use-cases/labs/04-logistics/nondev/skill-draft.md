# 업무 매뉴얼형 Skill 초안: 물류 배송 클레임·예외 처리 리포트

## 목적

배송 클레임을 유형·긴급도·담당 부서로 분류하고 일일 예외 처리 브리프를 만든다.

## 입력 자료

- 클레임 접수 CSV
- 미처리 배송 예외 목록
- 고객사 SLA 메모

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 내용을 분류합니다.
3. 근거가 부족하거나 승인이 필요한 항목을 `확인 필요`로 표시합니다.
4. 사람이 검토할 수 있는 표와 요약을 작성합니다.
5. 최종 공유 전 담당자가 표현과 민감정보를 확인합니다.

## 금지 사항

- 입력에 없는 사실을 만들어내지 않습니다.
- 보상·책임은 후보로만 표시한다. 기사명·연락처 등 개인 정보는 출력하지 않는다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
