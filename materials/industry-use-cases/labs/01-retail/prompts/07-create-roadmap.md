# Prompt 07: Create Roadmap

## 목적

1주 과정에서 파일럿 이후 4~8주 확장 Roadmap을 작성합니다.

## Agent 지시문

`docs/business-scenario.md`, `docs/task-breakdown.md`, `workspace/out/runbook.md`를 읽고 `workspace/out/roadmap.md`를 작성하세요.

Roadmap에는 다음 섹션을 포함하세요.

1. 1단계: 1개 업무 파일럿 안정화
2. 2단계: 인접 업무 확장
3. 3단계: 운영 지표와 검증 루틴
4. 역할과 책임(RACI 초안)
5. 리스크와 승인 게이트

확장 후보는 use case의 파일럿 확장 항목을 넘지 않게 작성하세요.
