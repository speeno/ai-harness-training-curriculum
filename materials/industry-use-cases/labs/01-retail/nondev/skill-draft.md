# 업무 매뉴얼형 Skill 초안: 유통 고객 VOC 분류·매장 주간 리포트

## 목적

채널별 VOC를 유형·긴급도·매장 영향으로 분류하고 주간 매장 이슈 리포트를 만든다.

## 입력 자료

- 고객센터 상담 메모
- 온라인몰 리뷰 CSV
- 매장별 개선 요청 메모
- 지난주 조치 현황

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 내용을 분류합니다.
3. 근거가 부족하거나 승인이 필요한 항목을 `확인 필요`로 표시합니다.
4. 사람이 검토할 수 있는 표와 요약을 작성합니다.
5. 최종 공유 전 담당자가 표현과 민감정보를 확인합니다.

## 금지 사항

- 입력에 없는 사실을 만들어내지 않습니다.
- 고객 이름·전화번호·주문번호는 출력하지 않는다. 매장 귀책은 근거가 있는 경우에만 후보로 표시한다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
