# Prompt 02: Refine Skill

## 목적

현재 샘플 데이터에 맞춰 Skill의 절차와 출력 계약을 점검합니다.

## Agent 지시문

`skills/launch-readiness-brief/SKILL.md`와 `docs/industry-constraints.md`를 읽고, `시즌 SKU 1건의 부서별 산출물을 모아 주간 런칭 준비 브리프를 만든다.`에 필요한 분류 기준·확인 필요 항목·출력 구조가 충분한지 검토하세요. 부족한 점이 있으면 `workspace/out/skill-review-notes.md`에 일반화된 개선안을 작성하세요. 단순히 이번 샘플만 맞추는 임시방편은 피하세요.
