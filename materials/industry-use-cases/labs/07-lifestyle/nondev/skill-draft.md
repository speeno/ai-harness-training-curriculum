# 업무 매뉴얼형 Skill 초안: 생활용품 통합 신제품 런칭 준비 브리프

## 목적

시즌 SKU 1건의 부서별 산출물을 모아 주간 런칭 준비 브리프를 만든다.

## 입력 자료

- 디자인 시안 피드백
- 상품 스펙·샘플 테스트 메모
- MOQ·단가 검토표
- 채널별 입고·프로모 일정

## 처리 절차

1. 입력 자료가 모두 있는지 확인합니다.
2. 업무 기준에 따라 내용을 분류합니다.
3. 근거가 부족하거나 승인이 필요한 항목을 `확인 필요`로 표시합니다.
4. 사람이 검토할 수 있는 표와 요약을 작성합니다.
5. 최종 공유 전 담당자가 표현과 민감정보를 확인합니다.

## 금지 사항

- 입력에 없는 사실을 만들어내지 않습니다.
- Go/No-Go를 자동 확정하지 않는다. 표시·인증·안전 관련 항목은 반드시 human_review_required로 표시한다.

## 출력 형식

- 요약
- 분류표
- 확인 필요 항목
- 다음 액션
